To handle update you can use the `hears`,`command`,`use`,`on` method. For example you can see this : 

## Using `hears` 
This method allow you to use regex or you can fill with some text 

<TabGroup>
    <TabContent name="javascript">
    ```ts
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() // fill the options. you can use the tgsnake.config.js file.
    // with text 
    bot.hears("hello",(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    // with regex 
    bot.hears(/hello/i,(ctx,match)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    bot.run() // running üêç
    ```
    </TabContent>

## Using `command`
This method only allow you to use string or array of string. The prefix is `.!/`  

<TabGroup>
    <TabContent name="javascript">
    ```ts
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() // fill the options. you can use the tgsnake.config.js file.
    // with text 
    bot.command("hello",(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    // with array
    bot.command(["hello","hi"],(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    bot.run() // running üêç
    ```
    </TabContent>

## Using `use`
This is a middleware. This function will execute first before other. 

<TabGroup>
    <TabContent name="javascript">
    ```ts 
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() 
    bot.use((ctx,next)=>{
        console.log(ctx)
        return next()
    })
    bot.run() // running üêç
    ```
    </TabContent>
    <TabContent name="typescript">
    ```ts 
    // index.ts
    import {Snake} from "tgsnake"
    const bot = new Snake() 
    bot.use((ctx,next)=>{
        console.log(ctx)
        return next()
    })
    bot.run() // running üêç
    ```
    </TabContent>
</TabGroup>
You can use the `Composer` as middleware. 
<TabGroup>
    <TabContent name="javascript">
    ```ts 
    // index.js
    const {Snake,Composer} = require("tgsnake")
    const bot = new Snake() 
    const startComposer = new Composer() 
    startComposer.command("start",(ctx)=>{
        ctx.reply("Welcome")
    })
    bot.use(startComposer.middleware())
    bot.run() // running üêç
    ```
    </TabContent>
    <TabContent name="typescript">
