Untuk menangani pembaruan, Anda dapat menggunakan metode `hears`,`command`,`use`,`on`. Misalnya Anda dapat melihat ini: 

## Menggunakan `hears` 
Metode ini memungkinkan Anda untuk menggunakan regex atau Anda dapat mengisi dengan text. 

<TabGroup>
    <TabContent name="javascript">
    ```ts
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() // fill the options. you can use the tgsnake.config.js file.
    // with text 
    bot.hears("hello",(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    // with regex 
    bot.hears(/hello/i,(ctx,match)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    bot.run() // running üêç
    ```
    </TabContent>

## Menggunakan `command`
Metode ini hanya memungkinkan Anda untuk menggunakan string atau larik string. Awalannya adalah `.!/`  

<TabGroup>
    <TabContent name="javascript">
    ```ts
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() // fill the options. you can use the tgsnake.config.js file.
    // with text 
    bot.command("hello",(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    // with array
    bot.command(["hello","hi"],(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    bot.run() // running üêç
    ```
    </TabContent>

## Menggunakan `use`
Ini adalah middleware. Fungsi ini akan dijalankan terlebih dahulu sebelum yang lain. 

<TabGroup>
    <TabContent name="javascript">
    ```ts 
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() 
    bot.use((ctx,next)=>{
        console.log(ctx)
        return next()
    })
    bot.run() // running üêç
    ```
    </TabContent>
    <TabContent name="typescript">

</TabGroup>
