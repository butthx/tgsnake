Untuk menangani pembaruan, Anda dapat menggunakan metode `hears`,`command`,`use`,`on`. Misalnya Anda dapat melihat ini: 

## Menggunakan `hears` 
Metode ini memungkinkan Anda untuk menggunakan regex atau Anda dapat mengisi dengan text. 

<Tab.Group>
    <Tab.List>
        <TabName>javascript</TabName> 
        <TabName>typescript</TabName>
    </Tab.List>
    <Tab.Panels> 
        <Tab.Panel>
```ts 
// index.js
const {Snake} = require("tgsnake")
const bot = new Snake() // isi opsi. Anda dapat menggunakan file tgsnake.config.js.
// with text 
bot.hears("halo",(ctx)=>{
  ctx.reply("Hai") 
  if(ctx.replyToMessage){ 
    ctx.replyToMessage.reply("Siapa kamu?")
  }
})
// dengan regex 
bot.hears(/halo/i,(ctx,match)=>{
  ctx.reply("Hai") 
  if(ctx.replyToMessage){ 
    ctx.replyToMessage.reply("Siapa kamu?")
  }
})
bot.run() // jalankan üêç
```
        </Tab.Panel>

## Menggunakan `command`
Metode ini hanya memungkinkan Anda untuk menggunakan string atau larik string. Awalannya adalah `.!/`

    <Tab.List>

    </Tab.List>
    <Tab.Panels>

// index.js
const {Snake} = require("tgsnake")
const bot = new Snake() // isi opsi. Anda dapat menggunakan file tgsnake.config.js.
// dengan text 
bot.command("halo",(ctx)=>{
  ctx.reply("Hai") 
  if(ctx.replyToMessage){ 
    ctx.replyToMessage.reply("Siapa kamu?")
  }
})
// dengan array
bot.command(["halo","hai"],(ctx)=>{
  ctx.reply("Hai") 
  if(ctx.replyToMessage){ 
    ctx.replyToMessage.reply("Siapa kamu?")
  }

## Menggunakan `use`
Ini adalah middleware. Fungsi ini akan dijalankan terlebih dahulu sebelum yang lain. 

<Tab.Group>
    <Tab.List>
        <TabName>javascript</TabName> 
        <TabName>typescript</TabName>
    </Tab.List>
    <Tab.Panels> 
        <Tab.Panel>
```ts 
// index.js
const {Snake} = require("tgsnake")
const bot = new Snake() 
bot.use((ctx,next)=>{
    console.log(ctx)
    return next()
})
bot.run() // jalankan üêç
```
