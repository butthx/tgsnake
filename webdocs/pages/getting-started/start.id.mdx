Hai! Apakah anda ingin membuat bot telegram dengan mtproto di javascript atau TypeScript? Jika ya tgsnake adalah framework yang terbaik untuk Anda. Secara singkat, tgsnake adalah framework mtproto yang menggunakan gramjs sebagai dasarnya. Kami telah menyederhanakan raw Api dari gramjs menjadi metode yang sederhana dan mudah dipahami.
## Mendapatkan ApiHash dan ApiId
Di mana saya bisa mendapatkan apiHash dan apiId?
Anda dapat menemukannya dari [my.telegram.org](https://my.telegram.org). Anda harus membuat aplikasi di sana. Peringatan! Jangan sebarkan ApiHash dan ApiId Anda!
## Setelah membuat aplikasi
Setelah membuat aplikasi, Anda membuat proyek di komputer Anda. Kemudian install framework `tgsnake`.
1. `mkdir tgsnake-project`
2. `cd tgsnake-project`
3. `npm init -y`
4. `npm i tgsnake --save`
5. `code .`
Kemudian buat file `index.js` dan mulailah menulis kode. contoh kodenya bisa anda lihat di [halaman pertama](/). Kemudian jalankan kode Anda.
## Bagaimana cara menghasilkan stringSession?
Untuk menghasilkan stringSession Anda dapat menggunakan metode `generateSession`.
   
Untuk pertanyaan lebih lanjut, Anda dapat bertanya dalam grup atau Anda dapat mengobrol dengan saya di telegram pribadi.
