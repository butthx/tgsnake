## Overview 
Sending sticker with fileId/file location/url/buffer. 
## Parameters 
```ts  
ctx.telegram.sendSticker(chatId,sticker)
```
- `chatId` (_number|string_) 
    - Chat/Groups/Channel id. 
- `sticker` (_string|Buffer_)
## Example 
```ts:index.ts {3} showLineNumber 
bot.on("message",async (ctx) => {
    if(ctx.media && ctx.media.type == "sticker"){
        let results = await ctx.telegram.sendSticker(ctx.chat.id,ctx.media.fileId) 
        console.log(results)
    }
})
```

Same as [sendMedia](/en/telegram/media/SendMedia#possibleerror)
