## Overview 
Sending Document file location/url/buffer. 
## Parameters 
```ts  
ctx.telegram.sendDocument(chatId,document,more)
```
- `chatId` (_number|string_) 
    - Chat/Groups/Channel id. 
- `document` (_string|Buffer_) 
    - File Location/Url/Buffer 
- `more` (_[sendMediaMoreParams](/en/telegram/media/SendMedia#parameters)_)
## Example 
```ts:index.ts {2} showLineNumber 
bot.command("doc",async (ctx) => {
    let results = await ctx.telegram.sendDocument(ctx.chat.id,"https://tgsnake.js.org/images/tgsnake.jpg")
})
```
## Possible Error 
Same as [sendMedia](/en/telegram/media/SendMedia#possibleerror)
