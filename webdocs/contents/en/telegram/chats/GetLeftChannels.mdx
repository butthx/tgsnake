## Overview 
Get a list of channels/supergroups we left. 
## Parameters 
```ts 
ctx.telegram.getLeftChannels(offset)
```
- `offset` (_number_) 
    - offset of pagination. 
## Example 
```ts:index.ts {2} showLineNumber 
bot.command("getLeftChannels",async (ctx) => {
    let results = await ctx.telegram.getLeftChannels() 
    console.log(results)
})
```
## Possible Error 
403 TAKEOUT_REQUIRED A takeout session has to be initialized, first
