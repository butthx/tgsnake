## Overview 
edit chat/group/channel title. 
## Parameters 
```ts 
ctx.telegram.editTitle(chatId,title) 
```
- `chatId` (_number|string_) 
    - Chat/Groups/Channel id. 
- `title` (_string_) 
    - New title 
## Example 
```ts:index.ts {3} showLineNumber 
bot.command("editTitle",async (ctx) => {
    if(!ctx.chat.private){
        let results = await ctx.telegram.editTitle(ctx.chat.id,"hey new title") 
        console.log(results)
    }
})
```
