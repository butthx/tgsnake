Handling Update.

To handle updates you can use 2 methods.

Using onNewMessage#

Only new message will receive with this method.

index.ts
bot.onNewMessage(ctx => {})

ctx is JSON update from telegram which we have added a prototype like ctx.reply

Using onNewEvent#

All event will receive with this method.

index.ts
bot.onNewEvent(event => {})

event is pure JSON update from telegram.
